### PiStreaming

## Requirement
- OpenCV
- Django (django-admin)
- PiCamera

## ReadMe
This is a simple Django Project. 
- Simple login custom
- Livestreaming from camera Pi
- Record function
- Save last 30s, 1m, 5m...
- Multiple threading
- Take a photo
- Modify resolution, framerate...
- Debug mode
- Media browser
